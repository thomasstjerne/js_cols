Class: Handle

js_cols.IntervalHeap. Handle

new Handle(k, v, i, c)

A handle used to locate a given element in the heap to obtain O(logn) running time for changeKey and remove. Handles are returned when inserting a key/value pair into the heap.

Parameters:
Name Type Description
k *

the priority key
v *

the value to insert
i Integer

the index in the heap
c js_cols.IntervalHeap

the heap containing the element referenced by this handle
