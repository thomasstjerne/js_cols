js_cols.js

Bootstrap for the js_cols Library. The code in this file, base.js, is adopted partly from googles base.js, and partly from structs.js. It has been modified to suit the needs of the js_cols Library. The js_cols collections should be easily integratable in projets using google closure library, as they generally use the same method names, code style and use of closures.
